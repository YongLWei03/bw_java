	添加：
		url:relatedunits
		method:post
		params:
		{
			"name":"上海市第一人民医院",
			"contact":"王梦飞",
			"tel":"13702569817",
			"unitType":1,
			"address":"徐汇区人民路001号",
			"sectionIds":"[1],[2],[3],[4]"  注意：前端传过来的值的格式必须符合这种要求，并且由于一个单位可能处于多个管廊的交叉口，因此添加的时候该项应该是多选框。
		}
		return:成功
	
	删除：
		url:relatedunits/{id}
		method:delete
		params:id
		return:成功
